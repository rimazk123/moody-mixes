# Moody Mixes

## Setup 
```bash
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
export SPOTIFY_CLIENT_ID={your_client_id}
export SPOTIFY_CLIENT_SECRET={your_client_secret}
```

## Run
```bash
python main.py
```
